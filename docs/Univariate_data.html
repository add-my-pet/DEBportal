Uni-variate data is a set of pairs of values for an independent and associated dependent variable. (AmP data types.)

Codification of types univariate data[edit]

In the mydata file the author needs to specify which types of data are used in the entry. Labels for uni-variate data are packed as metaData.data_1. The codification generally follows DEB's notation rules. The table below presents the labels for the different types of univariate data found in the collection (dark yellow cells to the far right of each row in the species list). For the labels the independent and the dependent variables are separated by "-".

Optionally this can be followed by an underscore ("_"), indicating several values of a modifying factor:

  • f, functional response
  • T, temperature
  • W, weight
  • L, length

Several f or T might mean several constant values, but also that they vary in time. Various labels relate to life history events, as listed for zero-variate data. Trailing e means that data refers to embryo development. Data might relate to males and females; this is not indicated in the data labels, but comments in such entries specify the parameters in which they differ. Filtering rate also includes food-searching rate (for species that don't filter at all).
